## 📂 Featured Projects

### 🐍 [VB.NET Snake Game](https://github.com/athronos21/VB.NET-Snake-Game)
> A fully functional classic Snake game built with Visual Basic .NET and Windows Forms
- 🎮 3 difficulty levels: **Easy**, **Medium**, **Hard**
- 🗺️ 3 game modes: **Full Border**, **Block obstacles**, **Full Open** (wrap-around)
- ⚡ Progressive speed increase every 5 food items eaten
- 🏆 Session-based high score tracking with "New High Score!" detection
- 🖼️ Animated directional sprites, GDI+ custom wall rendering
- **Stack:** `VB.NET` `Windows Forms` `.NET Framework` `GDI+`
